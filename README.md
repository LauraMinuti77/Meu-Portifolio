# Meu Portfólio

Este projeto é um site de portfólio desenvolvido durante atividades escolares, com o objetivo de praticar e aplicar conceitos básicos de desenvolvimento web.

## Sobre o Projeto
O site apresenta informações pessoais, uma breve descrição profissional e seções organizadas para demonstrar estudos e interesses na área de tecnologia.  
O foco do projeto é a prática de estruturação de páginas e estilização visual.

## Tecnologias Utilizadas
- HTML
- CSS

## Status
✅ Projeto concluído (atividade escolar)

## Créditos
Este projeto foi desenvolvido com base em um modelo de portfólio utilizado para fins educacionais e **adaptado por mim** durante meus estudos.

## Uso de Inteligência Artificial
Durante o desenvolvimento, foi utilizado o auxílio de ferramentas de Inteligência Artificial em alguns pontos, como apoio ao aprendizado, organização de ideias e ajustes de código.  
Todo o conteúdo foi analisado e adaptado para fins educacionais.
